# Startup
- Power up the [audio system][1]
- Power up the projector
- Power up the confidence monitor (big tv above door)
- Power up the [presentaion computer][2]
- Open NDI Screen Capture from taskbar
- Open [presenter][3] form taskbar
- Check **Configure Screens**n in Presenter to get the displays correct.
- Power up the [broadcast computer][4] **Pin 6580**
- Open [OBS][5] from task bar
- Power up the cameras on Stream Deck
- Distribute pastor's mic pack
- Distribute handheld mics. Orange to choir loft, Purple to mic stand by baptismal font
- Streaming setup
