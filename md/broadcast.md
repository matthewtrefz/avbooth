
# Broadcast On
- Power on the Broadcast PC
- Enter the 4 digit PIN **6580**. If an input box does not appear shake the mouse, or press the space bar. 
- From the task bar start OBS *Press OK if a plugin error message is dispalyed*
- Power on cameras
- Push the right arrow button on the Streamdeck to see the camera select menu 
- Left screen shows the video ging to the stream, and a control panel
- Right screen shows the four cameras. If this is not showing open the Multiview

# Broadcast Off
- Close all open apps on the Broadcast PC
- Power off cameras
- Shut down the Broadcast PC through the Start menu.
    - Start menu
    - Press the Power button
    - Press Shutdown from the pop up menu

# Camera setup
There are two ways to power the cameras on and off
1. Use the **Stream Deck**
 - You should see two buttons one for power on and off.
 - Push the camera power on button
2. Use the remote labeled Camera.
 - Point the remote at the cmaera and push the power button
 - repeat for each camera

## Camera Controls
To change the position of a camera
- Select a camera from the camera list
- Double click on the posiiton in the position list and the camera will move to that position
